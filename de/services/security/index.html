Security Compliance vom ersten Tag an!

Die Sicherheitsreife Ihrer Cloud Umgebung ist geschäftskritisch. Die Compliance ist für Unternehmen mit einer umfangreichen Applikationslandschaft eine grosse Herausforderung.

Cloud Security ist eine unserer Spezialitäten und wir unterstützen Sie gerne!

Kontakt aufnehmen

Wir sind spezialisiert auf

Compliant Architecture

Bei der Planung sicherheitskonformer Cloud Lösungen gibt es viele Faktoren zu berücksichtigen.

Wir bieten praxistaugliche Referenzarchitekturen, die CIS Control konform sind. Darüber hinaus prüfen wir gerne Ihre Architektur auf mögliche Schwachstellen.

Security Assessment

Wenn es um Security geht, ist eine andere Perspektive von einer unbeteiligten Stelle sehr wertvoll.

Wir führen gerne Sicherheitsaudits Ihrer Public Cloud Umgebung durch und helfen Ihnen, die gefundenen Mängel zu beheben.

Security Framework

Es gibt zahlreiche Cloud Security Frameworks (ISO 27000, CIS Controls, BSI C5, NIST) mit diversen Schwerpunkten.

Wir zeigen Ihnen, wie Sie die Compliance effizient überwachen und im Verletzungsfall angemessen reagieren können.

Security Finding Management

Ein Übermass an sicherheitsrelevanten Ereignissen kann Sie in einer kritischen Situation schnell handlungsunfähig machen.

Wir bieten eine cloud-native Lösung, um Ihre Security Finding unter Kontrolle zu bringen.

Image Lifecycle Management

Ein schlecht gehärtetes EC2-Image bietet eine grosse Angriffsfläche.

Wir unterstützen Sie bei der Implementierung eines Image-Backing Prozesses, der Ihr Unternehmen mit gehärteten EC2-Images versorgt und diese kontinuierlich validiert.

Logging und Monitoring

Das Sammeln von Logdaten in einer umfangreichen Cloud Umgebung kann schnell zu einer grossen Herausforderung werden.

Wir bieten diverse Optionen, um Logdaten sinnvoll zu sammeln und an eine Monitoring-Lösung Ihrer Wahl weiterzuleiten.

Schweizer Security Standard

Wir entwickeln alle unsere Lösungen mit dem Fokus auf Schweizer Security Standards.
Lösungen die den Sicherheitsanforderungen der Schweizer Finanzinstitute entsprechen, lassen wenig Raum für Schwachstellen. Bei uns sind Sie mit Ihren Sicherheitsanliegen goldrichtig!

AWS Security Baselining

Nicht nur die Workloads an sich, sondern auch die zugrunde liegende Cloud Foundation muss sicherheitskonform sein.
Wir zeigen Ihnen, wie Sie eine Reihe von grundlegenden Sicherheitskontrollen in Ihrer AWS Organisation definieren und implementieren. Mit vollständig sicherheitskonformen AWS Accounts lässt es sich nachts einfach besser schlafen.

AWS Security Architecture Review

Eine AWS Well-Architected Lösung sollte immer auf Ihre individuellen Sicherheitsanforderungen abgestimmt sein.
Es gibt zahlreiche Sicherheits-Benchmarks zur Bestimmung der Compliance-Rate. Wir können Ihnen bei der Entscheidung helfen, welche Kriterien berücksichtigt werden sollen. Gerne überprüfen wir auch Ihre Architektur auf mögliche Sicherheitslücken und stellen Ihnen Referenzarchitekturen zur Verfügung.
